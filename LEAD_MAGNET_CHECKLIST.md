### Page 2-3: Phase 1 ‚Äî Pre-Site Setup (4-6 Weeks Before)

**Connectivity Planning**

‚ñ° Survey site for mobile signal strength (all major networks)
‚ñ° Check fibre availability and lead times (usually 6-12 weeks)
‚ñ° Identify alternative connectivity options:
   ‚ñ° Starlink (fastest setup, works anywhere)
   ‚ñ° 4G/5G business broadband
   ‚ñ° Wireless point-to-point from nearby building
‚ñ° Order primary connectivity (minimum 50 Mbps for 20+ users)
‚ñ° Order backup connectivity (different technology from primary)
‚ñ° Arrange site visit from connectivity provider to check line of sight/obstacles

**Infrastructure**

‚ñ° Design site network layout (office locations, Wi-Fi coverage areas)
‚ñ° Plan cable routes (underground vs overhead, protection from vehicles)
‚ñ° Order network equipment:
   ‚ñ° Router/firewall (business-grade, not consumer)
   ‚ñ° Switches (with PoE for CCTV/phones if needed)
   ‚ñ° Wi-Fi access points (weatherproof for outdoor coverage)
   ‚ñ° Patch cables, trunking, cable management
‚ñ° Arrange power for network equipment (UPS backup recommended)
‚ñ° Plan equipment locations (secure, ventilated, accessible)

**User Accounts Planning**

‚ñ° Count total users (site team, office staff, subcontractors)
‚ñ° List required systems (email, file sharing, project management, etc.)
‚ñ° Prepare user onboarding spreadsheet (name, role, required access)
‚ñ° Set up email domain if new site (or confirm subdomain)
‚ñ° Configure Microsoft 365/Google Workspace licenses
‚ñ° Prepare standard user documentation (how to access email, Wi-Fi, etc.)

### Page 4-5: Phase 2 ‚Äî Site Setup Week (Week 1)

**Day 1-2: Connectivity**

‚ñ° Install and test primary internet connection
‚ñ° Install and test backup connection
‚ñ° Configure automatic failover between connections
‚ñ° Set up site Wi-Fi network (separate from guest network)
‚ñ° Test Wi-Fi coverage across entire site
‚ñ° Document network names and passwords securely

**Day 3-4: Core Systems**

‚ñ° Install and configure router/firewall
‚ñ° Set up network security (firewall rules, intrusion prevention)
‚ñ° Configure VPN for remote access
‚ñ° Set up file sharing (SharePoint, Google Drive, or server)
‚ñ° Test file access from multiple devices
‚ñ° Configure automated backups (local + cloud)
‚ñ° Test backup restoration (don't skip this!)

**Day 5: User Setup**

‚ñ° Create user accounts in batches by department
‚ñ° Configure email accounts
‚ñ° Set up multi-factor authentication (mandatory)
‚ñ° Install required software on site computers
‚ñ° Configure mobile devices (email, apps, security)
‚ñ° Print/email user guides with Wi-Fi passwords and login details
‚ñ° Hold 30-minute training session for site team

### Page 6: Phase 3 ‚Äî Security & Monitoring (Week 2)

**Essential Security**

‚ñ° Enable email security filtering (anti-phishing, anti-malware)
‚ñ° Configure endpoint protection on all computers
‚ñ° Set up mobile device management (MDM) for company phones
‚ñ° Enable encryption on laptops (BitLocker for Windows, FileVault for Mac)
‚ñ° Document password policy and share with all users
‚ñ° Enable audit logging on critical systems
‚ñ° Set up security alerts (failed logins, unusual activity)

**CCTV Setup (if applicable)**

‚ñ° Install cameras at key locations (gates, materials, site office)
‚ñ° Configure motion detection and alert zones
‚ñ° Set up remote access for project managers
‚ñ° Test night vision and image quality
‚ñ° Configure automatic cloud backup of footage
‚ñ° Document camera locations and coverage areas

**Access Control**

‚ñ° Set up door entry system (if applicable)
‚ñ° Configure access cards/codes for different user types
‚ñ° Set up visitor Wi-Fi (isolated from main network)
‚ñ° Document who has access to what systems

### Page 7: Phase 4 ‚Äî Project Management Tools (Week 2-3)

**Collaboration Setup**

‚ñ° Set up Microsoft Teams or Slack for site communication
‚ñ° Create channels for: Site Updates, Safety, Subcontractors, General
‚ñ° Configure file sharing integration
‚ñ° Set up video calling for remote meetings
‚ñ° Test audio/video quality from site office

**Project Management Software**

‚ñ° Configure Procore/Aconex/similar for document management
‚ñ° Upload current drawings and documentation
‚ñ° Set up user permissions (who can see/edit what)
‚ñ° Configure automated workflows (RFIs, submittals)
‚ñ° Train key staff on core features

**Mobile Access**

‚ñ° Install project apps on tablets/phones
‚ñ° Configure offline access for drawings
‚ñ° Test syncing when back online
‚ñ° Set up mobile printing if needed

### Page 8: Phase 5 ‚Äî Testing & Handover (Week 3)

**System Testing**

‚ñ° Test failover: unplug primary internet, verify backup takes over
‚ñ° Test backup restoration (restore a file, verify it's correct)
‚ñ° Test CCTV remote access from off-site location
‚ñ° Test VPN connection and file access
‚ñ° Simulate user lockout and recovery process
‚ñ° Test emergency contact procedures

**Documentation**

‚ñ° Create network diagram (simple sketch is fine)
‚ñ° Document all passwords in secure password manager
‚ñ° List all service providers with account numbers
‚ñ° Create "what to do if..." guide for common issues
‚ñ° Document escalation contacts (IT support, connectivity provider)

**User Acceptance**

‚ñ° Survey site team: "Is everything working? Any issues?"
‚ñ° Address any reported problems
‚ñ° Refine documentation based on feedback
‚ñ° Schedule weekly check-ins for first month

### Page 9: Phase 6 ‚Äî Ongoing (Monthly)

**Monthly Checks**

‚ñ° Review backup logs (ensure backups are running)
‚ñ° Test backup restoration (different file each month)
‚ñ° Review security alerts and logs
‚ñ° Check for software updates and apply
‚ñ° Review user accounts (remove leavers, add joiners)
‚ñ° Update documentation with any changes
‚ñ° Survey users for ongoing issues

**Quarterly Reviews**

‚ñ° Review connectivity performance (is bandwidth sufficient?)
‚ñ° Review security posture (any new threats to address?)
‚ñ° Plan for upcoming project phases (new requirements?)
‚ñ° Budget review (any upcoming renewals or investments needed?)

**Common Problems & Quick Fixes**

**Problem:** Can't connect to Wi-Fi
**Fix:** Check you're using the right network (not guest). Try forgetting the network and reconnecting.

**Problem:** Email not syncing
**Fix:** Check internet connection. Restart the app. Check account settings haven't changed.

**Problem:** VPN won't connect
**Fix:** Verify internet is working. Try different network. Contact IT for account check.

**Problem:** CCTV app not loading
**Fix:** Check internet connection. Try logging out and back in. Check for app updates.
