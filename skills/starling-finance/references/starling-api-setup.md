# Starling API Setup (User Checklist)

## What you need
- Starling Bank developer account
- A registered application with Client ID + Client Secret
- Access token (OAuth2)

## Steps (high level)
1) Create a Starling developer account
2) Register an application and obtain Client ID/Secret
3) Generate OAuth2 tokens (scopes for accounts + transactions)
4) Provide the credentials securely to this assistant

## Notes
- Keep tokens private
- Use read-only scopes unless write actions are explicitly required
