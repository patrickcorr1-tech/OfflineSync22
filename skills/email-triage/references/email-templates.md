# Email Templates for Follow-ups

Pre-written templates for different follow-up scenarios.

## Template Categories

### 1. General Follow-up (Professional)

```
Subject: Re: {original_subject}

Hi {name},

I hope this message finds you well. I wanted to follow up on my previous email regarding {topic}.

I understand things can get busy, but I wanted to make sure this didn't fall through the cracks. {specific_action}.

Please let me know if you need any additional information from me.

Best regards,
{signature}
```

### 2. Friendly Check-in

```
Subject: Re: {original_subject}

Hey {name},

Just circling back on this â€” wanted to make sure you saw my last message about {topic}.

No rush if you're swamped, but would love to hear back when you get a chance!

```

### 3. Urgent Follow-up

```
Subject: Re: {original_subject} - Follow-up Required

Hi {name},

I'm following up on my previous email from {days_ago} days ago regarding {topic}.

This requires your attention. {specific_action}.

Please respond by end of day if possible.

Thanks,
{signature}
```

### 4. Invoice/Payment Follow-up

```
Subject: Re: {original_subject} - Payment Status

Hi {name},

I hope you're doing well. I'm following up on the invoice I sent on {date} for {amount}.

I wanted to check in on the payment status. Please let me know if you need any clarification or if there's anything I can help with on your end.

If the payment has already been processed, please disregard this message.

Best regards,
{signature}
```

### 5. Meeting Request Follow-up

```
Subject: Re: {original_subject} - Scheduling

Hi {name},

I wanted to follow up on my request to schedule a meeting to discuss {topic}.

I'm available at the following times this week:
- {time_option_1}
- {time_option_2}
- {time_option_3}

Please let me know what works best for you, or suggest alternative times.

Looking forward to connecting,
{signature}
```

### 6. Proposal/Quote Follow-up

```
Subject: Re: {original_subject} - Next Steps

Hi {name},

I wanted to follow up on the {proposal/quote} I sent over {days_ago} days ago.

I understand these decisions take time. I'm happy to:
- Answer any questions
- Adjust the scope if needed
- Schedule a call to walk through details

What would be most helpful for you?

Best regards,
{signature}
```

### 7. No Response (Final Attempt)

```
Subject: Re: {original_subject} - Closing the Loop

Hi {name},

I've reached out a couple of times regarding {topic} but haven't heard back.

I understand priorities shift and this may no longer be relevant. I'll assume this is on hold unless I hear otherwise.

If you'd like to revisit this in the future, feel free to reach out anytime.

Best regards,
{signature}
```

## Placeholder Variables

| Variable | Description |
|----------|-------------|
| `{name}` | Recipient's first name |
| `{original_subject}` | Original email subject line |
| `{topic}` | Brief description of subject |
| `{specific_action}` | What you need them to do |
| `{days_ago}` | Number of days since last email |
| `{signature}` | Your email signature |
| `{date}` | Date of original invoice/email |
| `{amount}` | Invoice amount |
| `{time_option_X}` | Proposed meeting times |

## Tone Guidelines

**Professional:** Use for clients, senior contacts, formal business
**Friendly:** Use for colleagues, warm leads, existing relationships  
**Urgent:** Use sparingly for time-sensitive matters only
**Final Attempt:** Use after 2-3 follow-ups with no response

## Customization Tips

1. Always personalize the opening line
2. Reference specific details from their last message
3. Make the ask clear and specific
4. Keep it under 150 words when possible
5. Include a clear next step or question
